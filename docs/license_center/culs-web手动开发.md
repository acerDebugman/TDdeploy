



初始化 web 项目, 使用：

```
npx create-vue
```



```
cd culs-web
pnpm install
pnpm run format
pnpm run dev
```







```
# 1. 安装 Element Plus
pnpm install element-plus

# 2. 安装自动导入插件（推荐，按需加载，减小体积）
pnpm install -D unplugin-vue-components unplugin-auto-import
```







```

pnpm install js-cookie
pnpm install -D @types/js-cookie

pnpm install nprogress
pnpm install -D @types/nprogress


pnpm insall vue-i18n

pnpm add pinia-plugin-persistedstate
pnpm add -D pinia-plugin-persistedstate
```



重启 typescript server:

```
按 Ctrl + Shift + P → 输入 TypeScript: Restart TS Server
```
