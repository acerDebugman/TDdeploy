


ubuntu 安装 yarn:

```

# 1. 首先确保已安装 Node.js 和 npm
sudo apt update
sudo apt install -y nodejs npm

# 2. 使用 npm 全局安装 Yarn
sudo npm install -g yarn

# 3. 验证安装
yarn --version


```

这个才是真的可以按照某个版本执行的文档：

```

1. 拉取 docs 文档仓库 ，中文https://github.com/taosdata/docs.taosdata.com，英文https://github.com/taosdata/docs.tdengine.com 
2. 安装 yarn 
3. 同级目录放一个 tdengine 仓库，切换到你自己的测试分支 
4. 切换到文档仓库目录 
执行：
yarn install && yarn ass local  && yarn start   --host 0.0.0.0 -p 3000

5. 查看页面 localhost:3000/next , 比如 http://localhost:3000/next/reference/taos-sql/datain/， 而不是使用 http://localhost:3000/reference/taos-sql/datain/  这里 next 是预览版的区别

```
