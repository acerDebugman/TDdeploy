python 脚本导入：

```
import requests
import json
import time

url = "http://localhost:4242/api/put"
data = [
    {
        "metric": "cpu.idle",
        "timestamp": int(time.time()),
        "value": 99.0,
        "tags": {"host": "server1"}
    }
]

response = requests.post(url, data=json.dumps(data))
# 成功返回 204
print(response.status_code)
```
